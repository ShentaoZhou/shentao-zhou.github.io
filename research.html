SKY130 Bandgap Reference (BGR): Schematic → Layout → Verification Workflow

Designed and verified a curvature-compensated BGR in the open-source SKY130 PDK using Cadence Virtuoso. The workflow covers schematic design, full-custom layout, and DRC/LVS/PEX verification (e.g., Pegasus or compatible flows), followed by corner/Monte-Carlo simulations for linearity, noise, and temperature stability.

  • Focus: reference accuracy/TC, startup robustness, supply/PSRR and mismatch analysis.
  • Workflow: schematic → pre-layout sims → layout → DRC/LVS → PEX → post-layout sims.

TIA + MEMS Oscillator Interface

Built a transimpedance amplifier (TIA) to sustain a MEMS resonator, achieving stable oscillation and clock signal generation. The design balances noise, bandwidth, and startup/phase-noise performance, with biasing and loading tuned around the resonator characteristics.

  • Key concerns: input-referred noise, loop gain/phase, startup margin, and output buffering.
  • Verification: transient startup, phase-noise estimation, PVT robustness.

Auto-Cryo-Device-Model (Ongoing)

Building an automated modeling flow for low-temperature (cryo) CMOS devices, including data acquisition/curation, compact-model fitting, and ML-assisted surrogate modeling for model parameter extraction and circuit-level prediction. Goal: accelerate cryo-CMOS design/verification by streamlining device-to-circuit translation.

TMR-ADC: Tunnel Magnetoresistance Front-End + ADC Readout

Exploring a TMR (Tunnel Magnetoresistance) sensor front-end combined with an ADC readout chain. Emphasis on front-end SNR and linearity, reference stability, and system-level calibration for robust magnetic-to-digital conversion.
